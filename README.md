## Sunumu Açmak İçin

1. VS Code'da `PRESENTATION.md` dosyasını aç
2. Sağ üstteki **Marp** ikonuna tıkla
3. `Export Slide Deck` → PDF veya HTML olarak dışa aktar

Marp extension kurulu değilse:
```
VS Code → Extensions → "Marp for VS Code" ara → Kur
```
